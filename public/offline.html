📱 Apple iOS Guidelines:

  • 02. Tap the top Wi-Fi row element

💻 Windows Desktop Guide:

  • 01. Go to the desktop taskbar (bottom right monitor area)
  • 02. Click the Network/Wi-Fi Icon 📶

  • 04. Make sure you complete browser portal auth if on public coffee/hotel hotlines

📺 Smart TV & Remote Assistant:

  • 03. Choose Set up connection and key in router passcodes
